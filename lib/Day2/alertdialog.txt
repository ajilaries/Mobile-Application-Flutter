in flutter an AlertDialog is ued to show important messages , confirmations, warnings etc. 
We use showDialog() + AlertDialog() widget for creating alert dialog


Basic Example

showDialog(
  context: context,
  builder: (context) {
    return AlertDialog(
      title: Text("Alert"),
      content: Text("This is an alert dialog."),
      actions: [
        TextButton(
          onPressed: () {
            Navigator.pop(context); // closes dialog
          },
          child: Text("OK"),
        ),
      ],
    );
  },
);
